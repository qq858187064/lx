pause版本：CSS2　 兼容性：NONE　继承性：无

 
语法：
 
pause : pause-before || pause-after
 
参数：
 
该属性是复合属性。请参阅各参数对应的属性。
 
说明：
 
设置对象前后的声音暂停。
假如只给出了一个值，它将用于pause-before属性和pause-after属性。假如给出了两个值，则第一个用于pause-before属性，第二个用于pause-after属性。
目前IE5.5尚不支持此属性。
对应的脚本特性为pause。请参阅我编写的其他书目。
 
示例：
 
div { pause: 2s 3s }
div { pause: 2s; }
