color版本：CSS1　 兼容性：IE4+ NS4+　继承性：有

 
语法：
 
color : color
 
参数：
 
color : 　指定颜色。请参阅颜色单位和附录：颜色表
 
说明：
 
检索或设置对象的文本颜色。无默认值。
注意，用颜色名称指定color不被一些浏览器接受。
对应的脚本特性为color。请参阅我编写的其他书目。
 
示例：
 
div {color: #345456; }
div { color: rgb(100,14,200); }
div {color: menu; }
div {color: red; }
