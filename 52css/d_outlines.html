Outlines Properties
属性		CSS Version
版本		Compatibility
兼容性		Inherit From Parent
继承性		Description
简介
outlineCSS2NONE无复合属性。设置或检索对象外的线条轮廓
outline-colorCSS2NONE无设置或检索对象外的线条轮廓的颜色
outline-styleCSS2NONE无设置或检索对象外的线条轮廓的样式
outline-widthCSS2NONE无设置或检索对象外的线条轮廓的宽度

 
 
 
说明：这种颜色是CSS2标准属性。这种颜色是IE建议样式表属性。这种颜色是NS私有属性。这种颜色是目前尚无浏览器支持的属性。
