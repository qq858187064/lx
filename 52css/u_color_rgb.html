rgb(R,G,B)版本：CSS1　 兼容性：IE4+ NS4+

 
参数：
 
R : 　红色值。正整数 | 百分数
G : 　绿色值。正整数 | 百分数
B : 　蓝色值。正整数 | 百分数
 
说明：
 
以上三个参数，正整数值的取值范围为：0 - 255。百分数值的取值范围为：0.0% - 100.0%。超出范围的数值将被截至其最接近的取值极限。
并非所有浏览器都支持使用百分数值。
请参阅附录：颜色表。
 
示例：
 
div { color: rgb(132,20,180); }
div { color: rgb(12%,200,50%); }
