CSS Pseudo-Elements Reference    伪对象

 
Pseudo-Elements
伪对象		CSS Version
版本		Compatibility
兼容性		Description
简介
:first-letterCSS2IE5.5+设置对象内的第一个字符的样式表属性
:first-lineCSS2IE5.5+设置对象内的第一行的样式表属性
:beforeCSS2NONE用来和content属性一起使用，设置在对象前（依据对象树的逻辑结构）发生的内容
:afterCSS2NONE用来和content属性一起使用，设置在对象后（依据对象树的逻辑结构）发生的内容

 
 
说明：这种颜色是CSS2标准伪对象。这种颜色是IE建议样式表伪对象。这种颜色是NS私有伪对象。这种颜色是目前尚无浏览器支持的伪对象。
