outline-color版本：CSS2　 兼容性：NONE　继承性：无

 
语法：
 
outline-color : color |invert
 
参数：
 
color : 　指定颜色。请参阅颜色单位和附录：颜色表
invert : 　使用背景色的反色
 
说明：
 
设置或检索对象外的线条轮廓的颜色。参见outline属性。
目前IE5.5尚不支持此属性。
对应的脚本特性为outlineColor。请参阅我编写的其他书目。
 
示例：
 
img { outline-color: red }
p { outline-color: #E9E9E9 }
