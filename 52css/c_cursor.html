说明：
 
设置或检索在对象上移动的鼠标指针采用何种系统预定义的光标形状。
对应的脚本特性为cursor。请参阅我编写的其他书目。
 
示例：
 
p { cursor: text; }
a { cursor: pointer; }
body { cursor: url("mycursor.gif"), url("images/cursors/footcursor.jpg"), default; }
