zoom版本：IE5.5+专有属性　继承性：无

 
语法：
 
zoom : normal | number
 
参数：
 
normal : 　使用对象的实际尺寸
number : 　百分数|无符号浮点实数。浮点实数值为1.0或百分数为100%时相当于此属性的normal值
 
说明：
 
设置或检索对象的缩放比例。
对应的脚本特性为zoom。请参阅我编写的其他书目。
 
示例：
 
div {zoom : 0.75; }
 
您所看到的文字块的名字是小强，图片的名字是旺财，请您从下方的选择框内加减选择旺财和小强的zoom值，看一看会发生什么，然后您就会明白这个属性的意义。希望您喜欢这本电子书。谢谢。
