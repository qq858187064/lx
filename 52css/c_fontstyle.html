font-style版本：CSS1　 兼容性：IE4+ NS4+　继承性：有

 
语法：
 
font-style : normal | italic | oblique
 
参数：
 
normal : 正常的字体
italic : 斜体。对于没有斜体变量的特殊字体，将应用oblique
oblique : 倾斜的字体
 
说明：
 
设置或检索对象中的字体样式。
对应的脚本特性为fontStyle。请参阅我编写的其他书目。
 
示例：
 
p { font-style: normal; }
p { font-style: italic; }
p { font-style: oblique; }
