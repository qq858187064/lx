z-index版本：CSS2　 兼容性：IE4+ NS6+　继承性：无

 
语法：
 
z-index : auto | number
 
参数：
 
auto : 　遵从其父对象的定位
number : 　无单位的整数值。可为负数
 
说明：
 
检索或设置对象的层叠顺序。
如两个绝对定位对象的此属性具有同样的值，那么将依据它们在HTML文档中声明的顺序层叠
对应的脚本特性为zIndex。请参阅我编写的其他书目。
 
示例：
 
div { position:absolute; z-index:3; width:6px; }
